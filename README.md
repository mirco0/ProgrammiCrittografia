# Soluzioni CrittoEsercizi
In questa repository sono presenti le mie soluzioni agli esercizi periodici del corso di Crittografia, presso la facoltà di Informatica di TorVergata

## Come eseguire il codice
Ogni codice ha una documentazione (seppur minima), per ogni esercizio sono scritte le classi necessarie per la soluzione (Non essendo matematico molte classi che implementano concetti matematici sono ridotti al solo scopo implementativo). Il codice è generalmente eseguibile dal file principale che prende il nome dell'esercizio (per esempio `esercizio6.py`).
```bash
python3 "CrittoEsercizi2/Esercizio 6/esercizio6.py"
```

### Struttura della repository
Le consegne sono divise per cartelle, a ogni cartella una consegna. Nonostante il disordine, i diversi esercizi sono, a loro volta, divisi per cartelle in cui è presente il codice necessario per la risoluzione.

Inoltre per ogni consegna è disponibile il codice $\LaTeX$ (a meno del file ```matricola.tex```), consultabile liberamente in ogni sua parte.

Cercherò di mantenere la repository ordinata e coerente con la descrizione letta sopra. Alcuni file nel tempo potrebbero variare e non rispettare più il nome e/o contenuto originale. Ma proverò ad aggiornare anche i file latex per mantenere il tutto coerente.
